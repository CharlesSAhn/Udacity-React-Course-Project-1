#Udacity's React Fundamentals course Project #1## TL;DR

#Instruction for setup:
* `npm install` to install the required dependencies
* `npm start` to start the web app

## Source code layout
```bash
├── CONTRIBUTING.md
├── README.md - This file.
├── SEARCH_TERMS.md # The whitelisted short collection of available search terms for you to use with your app.
├── package.json # npm package manager file.
├── public
│   ├── favicon.ico # React Icon, You may change if you wish.
│   └── index.html # DO NOT MODIFY
└── src
    ├── App.css              # Styles for app.
    ├── App.js               # Root of this app.
    ├── ListBooks.js         #    Renders Header and Bookshelf
    ├── Bookshelf.js         #       Renders Books by shelf types
    ├── Books.js             #          Renders the book (thumbmail, title, authors) and the renders BookShelfChange
    ├── BookShelfChanger.js  #             Renders the selection for shelf update
    ├── SearchPage.js        #    Renders the search inputs and results via Books.js
    ├── App.test.js
    ├── BooksAPI.js # A JavaScript API for the provided Udacity backend.
    ├── icons # Helpful images for your app. Use at your discretion.
    │   ├── add.svg
    │   ├── arrow-back.svg
    │   └── arrow-drop-down.svg
    ├── index.css # Global styles. You probably won't need to change anything here.
    └── index.js # You should not need to modify this file. It is used for DOM rendering only.
```
